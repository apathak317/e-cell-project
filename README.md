# e-cell-project
## Smart waste management system
##  Project Overview
This project is a startup landing page developed as part of the E-Cell RGPV task.  
It presents a business idea called **Smart Waste Management System** which aims to solve the problem of inefficient waste collection in urban areas.
The website provides a structured overview of:
- Problem Statement  
- Proposed Solution  
- Market Research  
- Value Proposition  
- Business Model  
- Implementation Plan
##  Technologies Used
- HTML:Structure of the website  
- CSS : Styling and layout  
- JavaScript :Interactivity and dynamic behavior  
## How to Run the Project
1. Download or clone the repository  
2. Open the project folder  
3. Double-click on `index.html`  
4. The website will open in your browser
